# ISS Video Publishing Copy

Use this file for YouTube, LMS upload pages, internal release notes, or a product website resource center.

## Channel Positioning

Two series are recommended:

1. `ISS Quick Tours`
   - short marketing-oriented videos
   - objective: show value fast

2. `ISS Guided Tutorials`
   - longer operator-focused videos
   - objective: teach real usage

## Brand Description

`ISS is an ERP platform for operations, inventory, procurement, sales, service, finance, reporting, and administrative control. These videos show how each part of the system works in practice.`
